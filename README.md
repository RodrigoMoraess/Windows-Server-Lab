# Windows-Server-Lab
Active Directory + DHCP + Políticas de Segurança + Aplicação de Faturação

## Objetivo

Implementar e configurar um ambiente empresarial em Windows Server 2008,
incluindo Active Directory, DHCP, políticas de segurança e auditoria,
além da instalação de uma aplicação de faturação em ambiente de rede.

---

## Tecnologias

- Windows Server 2008
- Active Directory
- Servidor DHCP
- Políticas de Segurança e Auditoria
- Pastas Partilhadas em Rede
- Aplicação de Faturação Empresarial

---

## Visão Geral do Ambiente

O laboratório simula um servidor empresarial responsável por:

- Gestão de utilizadores e permissões
- Serviços de rede (DHCP)
- Políticas de segurança e auditoria
- Implementação de aplicações partilhadas
- Controlo centralizado de acesso
